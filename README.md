# 🛒 React E-Commerce Website

A modern and responsive **E-Commerce web application** built using **React.js**.
This project allows users to browse products, view details, add items to cart, and manage their shopping experience smoothly.

---

## 🌐 Live Demo

👉 https://frontend-project-gules-seven.vercel.app/

---

## ✨ Features

* 🛍️ Browse products from FakeStore API
* 🔍 View detailed product information
* 🛒 Add to Cart functionality
* ➕ Increase / decrease product quantity
* ❌ Remove items from cart
* 💰 Dynamic price calculation (subtotal, tax, total)
* 🚚 Shipping calculation
* 📱 Fully responsive UI
* ⚡ Fast performance with Vite

---

## 🧠 Tech Stack

* **Frontend:** React.js
* **Styling:** Tailwind CSS
* **State Management:** Context API + useReducer
* **Routing:** React Router
* **API:** FakeStore API
* **Build Tool:** Vite
* **Deployment:** Vercel

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/snehagupta236/frontend-project.git
```

Navigate to project folder:

```bash
cd frontend-project/e-commerce
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```
